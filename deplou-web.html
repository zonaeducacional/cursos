🚀 Deploy da EduPlatform - Sem npm

📋 Pré-requisitos

✅ Projeto Firebase: cursos-online-5aaac

✅ Credenciais configuradas no código

🔥 1. Configurar Firebase Console

  1. Acesse: Firebase Console
  2. Authentication → Sign-in method → Ativar Google
  3. Firestore Database → Criar banco → Modo produção → Localização: southamerica-east1
  4. Hosting → Vamos começar

🌐 2. Deploy via Netlify (Recomendado)

  1. Acesse: netlify.com
  2. Faça cadastro gratuito
  3. Arraste a pasta public para a área de deploy
  4. Aguarde o deploy (1-2 minutos)
  5. Copie a URL gerada
Vantagem: Deploy em segundos, HTTPS automático, domínio gratuito

🐍 3. Teste Local (Python)

cd public
python3 -m http.server 8000

Acesse: http://localhost:8000

👑 4. Configurar Admin

  1. Acesse sua plataforma (Netlify ou local)
  2. Faça login com Google
  3. No Firestore Console
  4. Encontre seu usuário na coleção users
  5. Edite o campo role para "admin"

🎯 5. Testar Funcionalidades

  • ✅ Login com Google
  • ✅ Acesso ao painel admin: /admin/
  • ✅ Criar primeiro curso
  • ✅ Dashboard estudante: /dashboard/

⚠️ Problemas Comuns

  • Erro de CORS: Use HTTPS (Netlify resolve automaticamente)
  • Login não funciona: Verificar se Google Auth está ativo
  • Firestore erro: Verificar se banco foi criado

🎉 Sucesso!

Sua plataforma estará funcionando em:

  • Netlify: https://seu-site.netlify.app
  • Local: http://localhost:8000
